# Сайт-калькулятор
В данном сайте возможно вычисление двух чисел с помощью их ввода и выбора операции: добавление, вычитание, умножение и деление. Сайт состоит из трёх файлов:
## index.html
В данном файле пропечатаны основной скелет сайта и форма вычисления.
## styles.css
В этом файле прописаны правила для стиля основных элементов, пропечатанных в скелете.
## app.js 
В этом файле пропечатана функция вычисления двух чисел выбранной операцией, и она будет вызвана кнопкой "Высислить" внутри сайта.
